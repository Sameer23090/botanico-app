**Botanico** is a high-performance, scientific plant tracking platform designed for researchers, commercial growers, and elite hobbyists. It features a stunning "Twilight Emerald" glassmorphism UI and integrates cutting-edge AI via **Groq LLaMA 3.1 & 3.2** for biometric health analysis and automated botanical documentation.

---

## ✨ Core Intelligence Layers

### 🧠 BotaniBot AI Consultant
A global, floating AI assistant powered by **LLaMA 3.1 70B**. BotaniBot maintains conversation context to provide real-time expert advice on soil science, pest management, and growth optimization.

### 🔬 Biometric Health Scan
Upload a photo during any care log update to trigger an **AI Health Scan**. Using **LLaMA 3.2 Vision**, Botanico identifies visible symptoms, assesses vitality (Excellent to Critical), and provides data-driven treatment protocols.

### 📊 Automated Botanical Insights
The Dashboard features a daily refreshed "Botanical Insight" generated by AI, providing scientifically accurate plant facts and care tips tailored to the professional botanist.

### 🛒 Marketplace Listing Generator
Create professional listings in seconds. The AI analyzes your plant species and category to generate compelling, SEO-optimized descriptions for the Marketplace exchange.

---

## 🛠️ Features & Precision

- **Elite Biotech UI**: A premium "Twilight Emerald" dark-mode interface using glassmorphism, fluid motion (Framer Motion), and a custom liquid cursor.
- **Scientific Logging**: Track taxonomic data (Binomial nomenclature), growth metrics (height, leaf count), and phenological stages.
- **Mission Control (Reminders)**: Sophisticated care scheduling with priority tiers, frequency control, and overdue detection.
- **Marketplace Exchange**: A complete peer-to-peer exchange for seeds, equipment, and live specimens with filtered search.
- **Public QR Profiles**: Generate unique QR codes for every plant that link to a public-facing, unauthenticated "Specimen Profile" for easy physical tracking.

## 📦 Setup & Deployment

### Environment Variables Required

#### Backend (`backend/.env`)
```env
PORT=5000
MONGODB_URI=mongodb+srv://...
JWT_SECRET=your_jwt_secret
GROQ_API_KEY=gsk_...
GOOGLE_SERVICE_ACCOUNT_KEY_JSON={...}
DRIVE_ROOT_FOLDER_ID=...
```

#### Frontend (`frontend/.env.local`)
```env
VITE_API_BASE_URL=https://your-backend.vercel.app/api
```

### Quick Start

1. **Clone & Install**
   ```bash
   git clone https://github.com/Sameer23090/botanico-app.git
   cd botanico-app
   npm install && cd frontend && npm install && cd ../backend && npm install
   ```

2. **Run Locally**
   ```bash
   # Terminal 1: Backend
   cd backend && npm run dev
   
   # Terminal 2: Frontend
   cd frontend && npm run dev
   ```

3. **Build for Production**
   ```bash
   cd frontend && npm run build
   ```

## 🛡️ Security Hardening

- **SSRF Protection**: Strict whitelist for AI image analysis hosts.
- **Environment Aware**: No hardcoded `localhost` references; dynamic API routing for Vercel production.
- **Sanitized Inputs**: Zero-dependency on client-side state for sensitive DB operations.
- **Private Secrets**: All API keys and Database URIs are excluded from version control.
